# Barkin!

**BLUF**: This a remake of my internship project!

Why? It initially used a really limited and new "programming language" that, while was really fun to push, held back certain aspects and was a pain to use. Additionally, I wanted more practice with web development (at least using Flask and HTML/CSS, the JS is intimidatingly next), so knowing everything that I've learned since then, I thought it'd be fun to redo.

The goal of my internship project was to create a database and database application to log analyst reviews of courses, workshops, conferences, etc. The goal was to allow analysts to make more informed decisions about continuing their professional development that take into account certain factors- especially budget and time. They could submit, search through, filter, or get more details about reviews.

This version is ultimately pretty much the same other than the huge fact that it was made with Python/Flask and HTML (and not to mention SQL, can you believe I really had to do all that without SQL before??). There are a few additions, most notably an improved/more efficient search and filter, better distinction between event and review entities, ability to create an account, having the option to save events, and most drastic of all, it is now blue instead of green...

It's also dog themed this time :) woof

There are definitely many improvements I could make, and I might come back in the future to improve or redo it (actually use JS to make it look nicer, implement admins, implement deleting/editing reviews, etc.), but I think it's decent given I'm still kinda new-ish Flask and especially HTML/CSS. I also just want to work on other projects now... woof.

## To run this, you just need to clone the repo, install the requirements, and run the following commands:  
`flask --app barkin_site init-db`  
`flask --app barkin_site run`  
### Or, alternatively, just go here: https://barkin.onrender.com/
(assuming it's up)
